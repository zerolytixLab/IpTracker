[+] Features
🌍 Geolocation tracking

🏢 Network intelligence (ISP, Organization, ASN)

🚫 Proxy/VPN detection

📱 Mobile network detection

📊 Save reports as TXT

🎯 Support multiple IPs

[+] Installation
bash
git clone https://github.com/zerolytixLab/IpTracker.git
cd IpTracker
pip install -r requirements.txt
python iptracker.py
[+] Usage
bash
python iptracker.py -t 8.8.8.8
python iptracker.py -t 8.8.8.8 -o report.txt
